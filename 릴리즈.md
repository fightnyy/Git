# 릴리즈 하기

* 프로그램 버전 : 의미 있는 특정 시점의 맥락

- 메이저 업그레이드: 사용자들이 크게 느낄 변화를 적용 했을 때

- 마이너 업그레이드:작은 변화 등이 생겼을 때

  __Tip__ : LTS(Long Time Support)는 일반 버전보다 장기간에 걸쳐 지원하도록 특별히 만들어진 버전을 의미함

  ​	   : 예를 들어 10.15.0 이라는 버전명이 있으면 메이버 버전인 10, 마이너 버전인 5 그리고 maintenance 버전으		  로 0 버그나 작은 수정이 들어 갔을 때 바꿉니다.



* 릴리즈 : 프로그램을 출시하는 것
* 태그 : 특정 커밋에 포스트잇을 붙이는 느낌
* 소스트리에서 하는법
  1. 태그 아이콘에 우클릭을 누른뒤 새 태그 추가를 누른다.
  2. 태그 이름을 붙이고 추가를 누른다. 
  3. 태그가 원격저장소에는 push가 안되어 업데이트 되어있지 않으니 push를 눌러 업데이트 해준다.
  4. Github에 보면 release에 내가 한 버전이름 명이 보인다. Zip file로 다운받을수 있게 되어있다.
